# NVFP4 experiment record

The source tracks `scripts/quantize.sh`, `perf.sh`, `eval.sh`, `run_humaneval.sh`, `serve.sh`, `pipeline.sh`, release fixtures for BF16/NVFP4, `render_card.py`, and shell tests.

Those fixtures establish record shape and release-check expectations. They do not prove current speed, memory, quality, or reliability. No model files, private paths, credentials, host identifiers, or raw workload telemetry are published here; the public JSON is a non-live schema fixture.
